# Марк — Telegram Bot

## Установка за 5 минут

### 1. Создай бота в Telegram
1. Открой @BotFather в Telegram
2. Напиши `/newbot`
3. Придумай имя и username (например `mark_plan_bot`)
4. Скопируй токен — он выглядит так: `123456789:AAFxxx...`

### 2. Задеплой на Railway (бесплатно)

1. Зайди на [railway.app](https://railway.app) — войди через GitHub
2. Нажми **New Project → Deploy from GitHub repo**
3. Загрузи эти файлы в GitHub репозиторий (или используй Railway CLI)
4. В настройках проекта добавь переменную окружения:
   ```
   BOT_TOKEN = твой_токен_от_BotFather
   ```
5. В разделе **Settings → Deploy** укажи команду запуска:
   ```
   python bot.py
   ```

### 3. Запусти бота
Открой бота в Telegram и напиши `/start`

## Что умеет бот

- **📅 Сегодня / Завтра** — расписание с чекбоксами, автоматически меняется под тип дня
- **🗓 Календарь** — 7 дней вперёд, нажимаешь на день — открывается расписание
- **📖 Методичка** — все фазы обучения с задачами и ссылками, система зависимостей
- **🔥 Привычки** — сетка по неделям, навигация назад/вперёд
- **📊 Статистика** — стрик, сессии по категориям, фазы пройдено

## Добавление событий

Нажимаешь **➕ Добавить событие** в любом дне → выбираешь тип → пишешь описание.

Если в описании есть время (например "Оркестр 9:30"), блок автоматически появится в расписании.

## Данные

Хранятся в `data.json` рядом с ботом. На Railway данные сохраняются между перезапусками.

## Локальный запуск (для теста)

```bash
pip install -r requirements.txt
BOT_TOKEN=твой_токен python bot.py
```
